# dat.GUI

Haxe externs for dat.GUI

Source: https://github.com/dataarts/dat.gui/

API: https://github.com/dataarts/dat.gui/blob/master/API.md

Examples: http://workshop.chromeexperiments.com/examples/gui/#1--Basic-Usage

## vars convert to

- boolean => checkbox
- int/float => slider
- string => text input
- function => button
